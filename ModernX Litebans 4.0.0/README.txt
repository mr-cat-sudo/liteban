How to install this template:

1. Install litebans webinterface and connect it to your litebans plugin (https://www.spigotmc.org/resources/litebans.3715/)
2. Upload bans.zip to you file manager and extract files (Overwrite duplicate files)
3. Logo can be changed: open folders 'bans > inc > img', remove logo.png, upload your own logo and rename it to logo.png
4. Website icon can be changed: open folders 'bans > inc > img', remove favicon.ico, upload your own icon and rename it to favicon.ico
5. For all template settings open 'bans > inc > template.php'.
6. When on the webpage press CTRL + F5 and if you're using Cloudflare purge cache.

DONE!
